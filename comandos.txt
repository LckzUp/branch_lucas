Comandos usados para realizar a atividade do dia 14/03/2025

-Utulizados comandos cd (branch_lucas) para entrar nas pastas 
-Ao entrar na pasta usamos:
    git init

    git remote add origin https://github.com/LckzUp/branch_lucas.git

    git branch -m main

    git status

    git add .

    git status

    git commit -m "Criação arquivos"

    git log

    git push -u origin main (para colocar token)
